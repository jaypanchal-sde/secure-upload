# Secure Upload

Encryption-first file uploads for Node.js applications.

Secure Upload is a lightweight, storage-agnostic SDK that encrypts files before they are stored. It integrates seamlessly with Local Storage, AWS S3, Cloudflare R2, and custom storage providers, making secure file uploads simple and flexible.

---

## Features

* Encrypt files before storage
* Storage-agnostic architecture
* Support for Local Storage
* AWS S3 integration
* Cloudflare R2 integration
* Custom storage drivers
* Lightweight and performant
* TypeScript support
* Simple and extensible API

## Why Secure Upload?

Most upload libraries store files exactly as they are received. Secure Upload encrypts files before they are written to storage, ensuring your data remains protected regardless of where it is stored.

Its storage-agnostic design lets you switch providers without changing your application logic.
